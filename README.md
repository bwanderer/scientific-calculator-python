# Scientific Calculator — Python

A collection of Python scripts that evaluate and solve real-world scientific and engineering formulas from scratch.

## What it does

- **Euler's Number Approximation** — evaluates `(1 + 1/x)^x` across increasing values of x to demonstrate convergence toward *e* (~2.718)
- **Physics & Statistics Calculator** — computes Ohm's Law, kinetic energy, standard deviation (implemented manually without a library), and gravitational force
- **Engineering Formulas** — calculates Reynolds number (fluid dynamics), wave diffraction wavelength, hyperbolic oil production decline rate, and the Tsiolkovsky rocket equation (Δv)

## What it demonstrates

- Mathematical reasoning and formula implementation in Python
- Use of `math` and `statistics` standard libraries
- Loops, f-strings, and formatted output
- Implementing algorithms from scratch (e.g. standard deviation without `stdev()`)

## How to run

```bash
python euler_approximation.py
python physics_calculator.py
python engineering_formulas.py
```

Requires Python 3.x — no external libraries needed.

## Technologies

Python 3 · `math` · `statistics`
